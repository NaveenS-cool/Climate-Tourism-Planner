# Climate-Tourism-Planner
A Streamlit-based web application that analyzes climate suitability for tourism destinations using real-time weather and historical climate data, for more informed tourism planning.

## Features

- Search destinations across Kerala
- Automatic geocoding using Nominatim
- Real-time weather analysis
- Historical climate analysis using a 5-year seasonal window
- Tourism Climate Index (TCI) calculation
- Climate anomaly detection using Z-score analysis
- AI-powered travel recommendations using Groq Llama
- Clean and responsive Streamlit interface

## 🛠️ Tech Stack

### Frontend

- Streamlit

### Backend

- Python

### APIs

- OpenStreetMap Nominatim API
- Open-Meteo Forecast API
- Open-Meteo Archive API
- Groq API (Llama 3.1)
